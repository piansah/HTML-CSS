The Islands

Indonesian is an archipelago country that has more than 17.000++ island and 62% of sea area.

There are 7 main island, Sumatra, Kalimantan, Jawa Sulawesi, Papua, Kepulauan Maluku, Kepulauan Sunda Kecil.
